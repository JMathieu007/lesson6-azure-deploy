Deployment pipeline at a glance

1
Code push to GitHub
Joshua updates the project and pushes a commit to the main branch.
2
GitHub Actions workflow
A YAML workflow runs: checks out the code, sets up tooling, and calls Terraform.
3
Terraform apply
Terraform plans and applies the desired state: resource group, app service, and config.
4
Azure App Service
The built static files are deployed, and this page becomes available on the internet.
